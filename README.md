# Trouvaille

Trouvaille es una tienda online desarrollada con JavaScript enfocada en la experiencia de usuario, permitiendo explorar productos, filtrarlos y gestionar un carrito de compras de forma dinámica.

## Demo

🔗 https://trouvailletest.netlify.app/

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- SweetAlert
- Git
- GitHub

## Funcionalidades

- Visualización dinámica de productos.
- Filtrado por categorías.
- Carrito de compras interactivo.
- Persistencia de datos mediante LocalStorage.
- Cálculo automático de cantidades y totales.
- Interfaz responsive para distintos dispositivos.

## Lo que practiqué

- Manipulación del DOM.
- Consumo y gestión de datos en JavaScript.
- Eventos y renderizado dinámico.
- Almacenamiento local con LocalStorage.
- Organización de lógica y componentes frontend.

- Pasarela de pagos.
- Panel de administración de productos.
